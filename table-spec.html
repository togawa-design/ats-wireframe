グループ別テーブル数

グループテーブル概要備考
A. 認証・ログ Staff / StaffSession / AuditLog（3） Google OAuthによるスタッフ認証・セッション・操作監査 StaffSession・AuditLogは今回新規追加
B. 応募者 Applicant / ApplicantSource / ApplicantCustomField / ApplicantTag / Tag / School / LineRegistrationToken（7） 応募者情報・流入元・カスタム項目・タグ付け・LINE連携トークン sf_contact_idでSalesforce連携。カスタム項目はJSONBから正規化テーブルに変更
C. 選考フロー SelectionFlow / SelectionStage / Selection / SelectionHistory / Interview（5） 選考フロー設定・ステージ管理・面接記録 Interview.interviewer_id・google_calendar_event_idは今回追加
D. カレンダー CalendarSlot（1） 面接・説明会の枠管理。Google Calendarと双方向同期 google_synced_atは今回追加
E. LINE RichMenu / RichMenuRule / Survey / SurveyQuestion / SurveyResponse / SurveyResponseItem / LineMessage（7） LINEリッチメニュー管理・アンケート機能・個別チャット履歴 LIFFアプリと連携。Survey系はJSONBから正規化テーブルに変更
F. 媒体・マスター Media / QRCode / Document / RecruitmentGoal（4） 求人媒体・QRコード・書類テンプレート・採用目標のマスター管理 RecruitmentGoalは分析画面（SCR-06）の達成率算出に使用
